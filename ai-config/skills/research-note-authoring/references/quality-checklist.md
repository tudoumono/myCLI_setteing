# 完了前チェックリスト

## 分類

- `一般資料` / `PJ特化ノート` の振り分け理由が説明できるか
- PJ固有情報を `一般資料` に混ぜていないか

## 内容品質

- 先に結論があるか
- 目的・前提・対象読者が分かるか
- 比較や選択肢に判断理由があるか
- 未確認事項が明示されているか（ある場合）

## 出典

- 最新性が必要な情報は確認済みか
- 参考リンクまたは出典の記載があるか
- PJ特化ノートでユーザー起点の発想が埋もれていないか

## 構造

- 目次が必要な長さなら目次があるか
- 見出し階層が崩れていないか
- ファイル名が `NN_タイトル.md` 形式か（README除く）

## リンク

- 上部の README 戻りリンクが正しいか
- 相対リンクが切れていないか
- `scripts/validate_note_links.py` を実行したか（可能なら）

## README索引

- 新規/更新ノートが README から辿れるか
- 手書き説明を壊していないか
- 自動索引ブロックを使う場合、意図どおり更新されているか
